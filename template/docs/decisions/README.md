## Como abrir nova ADR

Rode `/adr <descrição em 1-2 linhas>` no Claude Code. O slash command guia a estruturação.

## Status válidos

- **proposed** — em discussão. Não vincula ainda.
- **accepted** — vincula. Hook ativo enforça quando aplicável (via `rules.yaml`).
- **superseded** — substituída por outra ADR. Mantida no histórico.
