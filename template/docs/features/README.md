## Como começar

Rode `/feature-breakdown` no Claude Code (precisa de PRD + personas prontos).

O comando vai:

1. Perguntar quais capacidades de alto nível o produto tem (3-7 itens).
2. Categorizar cada uma como `MUST` ou `OUT-OF-SCOPE`.
3. Identificar a **feature-âncora** aplicando 4 critérios canônicos.
4. Gerar tabela de features + diagrama de dependências + anti-features.

Depois, para cada `MUST`, rode `/feature-spec F<N>` para detalhar.
